# CompTIA Security+ SY0-701 Study Repository

This repository organizes my CompTIA Security+ SY0-701 preparation and connects exam objectives to hands-on cybersecurity practice.

## Professional Context

I am an aspiring Cybersecurity Analyst with 10+ years of IT experience across system administration, network management, and technical support. I am ISC2 Certified in Cybersecurity (CC), preparing for CompTIA Security+ SY0-701, and building practical labs in SOC triage, SIEM detection, incident response, vulnerability management, IAM, cloud security, and GRC.

| Domain | Focus |

| Security Program Management | Risk, governance, compliance, policy, audits, and third-party management |
