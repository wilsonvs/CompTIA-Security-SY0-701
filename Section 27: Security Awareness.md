## Security Awareness

- **Security Awareness:** Training and communication that helps users recognize threats and follow secure behavior.
- People are often targeted because attackers know human mistakes can bypass technical controls.

## Common Awareness Topics

- Phishing
- Password safety
- MFA use
- Data handling
- Safe browsing
- Removable media
- Physical security
- Remote work security
- Incident reporting
- Clean desk practices
- Acceptable use

## Social Engineering

- **Social Engineering:** Manipulating people into giving access, information, money, or action.

Warning signs:
- Urgent pressure
- Request for secrecy
- Unexpected attachment
- Suspicious link
- Unusual sender address
- Request for password or MFA code
- Gift card or payment request
- Impersonation of executive, vendor, or IT support

## Phishing Defense

Users should:
- Check sender address
- Be careful with attachments
- Hover over links when possible
- Watch for spelling or domain tricks
- Avoid entering credentials from unexpected links
- Report suspicious messages

## Remote Work Awareness

Remote work risks include:
- Public Wi-Fi
- Shared devices
- Shoulder surfing
- Lost or stolen laptops
- Weak home router security
- Unapproved cloud storage

Good practices:
- Use VPN when required
- Lock screen
- Use MFA
- Keep devices updated
- Protect company data
- Report lost devices quickly

## Reporting

Users should know how to report:
- Suspicious email
- Lost device
- Suspected credential compromise
- Unusual system behavior
- Accidental data exposure

## Key Takeaways

- Awareness helps users recognize and report threats early.
- Security culture improves when reporting is simple and encouraged.
